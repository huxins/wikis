# 参数绑定

方法参数名

```java
@RequestMapping("/hello")
public String hello(@RequestParam(value = "userId", required = false, defaultValue = "0") int id) {
    return String.valueOf(id);
}
```

路径

```java
@RequestMapping("/hello/{userId}")
public String hello(@PathVariable("userId") int id) {
    return String.valueOf(id);
}
```
