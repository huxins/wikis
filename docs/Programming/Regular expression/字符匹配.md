# 字符匹配

> word：数字、字母
>
> white space：空格、Tab
>
> any：任意字符

| 字符       | 说明               |
| ---------- | ------------------ |
| \d         | 匹配数字           |
| \w         | 匹配 word          |
| \W         | 匹配非 word        |
| \s         | 匹配 white space   |
| \S         | 匹配非 white space |
| .          | 匹配 any           |
| *          | 0，more            |
| +          | 1，more            |
| ?          | 0，1               |
| {min, max} | 范围               |
| {n}        | 指定               |
| ^          | 开头               |
| $          | 结尾               |
| \b         | word bounds        |

> 字符分类

- 分类符：`[]`，内部字符仅代表字符本身，表示逻辑关系`或`，另一种方法为 `(a|b)`。
- 连接符：`-`，在首位为字符本身，非首位表示范围。`[-.]` `[a-z]`
- 取反符：`^`，在首位为取反，非首位表示本身。
