| 语法 | 说明                                |
| ---- | ----------------------------------- |
| \*   | 通配符，默认附加两端                |
| \|   | 终止符，支持 HTTP 和 HTTPS 完整 URL |
| @@   | 例外规则                            |
| \|\| | 限定域名，包括 HTTP 和 HTTPS        |
| !    | 注释符                              |
| ^    | 占位符                              |
