## BIOS

### 戴尔

| 快捷键 | 说明         |
| ------ | ------------ |
| F2     | 设置 BIOS    |
| F12    | 启动设备选项 |

### 惠普

| 快捷键 | 说明         |
| ------ | ------------ |
| Esc    | 启动菜单     |
| F1     | 系统信息     |
| F2     | 系统检测     |
| F9     | 启动设备选项 |
| F10    | 设置 BIOS    |
| F11    | 系统恢复     |
| F12    | 网络启动     |
