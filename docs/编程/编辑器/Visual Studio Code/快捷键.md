# 快捷键

| NO   | 快捷键        | 描述             |
| ---- | ------------- | ---------------- |
| 1    | ctrl+k ctrl+c | 单行注释         |
| 2    | ctrl+k ctrl+u | 取消单行注释     |
| 3    | alt+shift+a   | 多行注释         |
| 4    | ctrl+b        | 侧边栏可见性     |
| 5    | ctrl+shift+k  | 删除当前行       |
| 6    | ctrl+~        | 控制台终端可见性 |
| 7    | ctrl+p        | 查找文件         |
| 8    | alt+shift+f   | 代码格式化       |
| 9    | ctrl+k ctrl+f | 格式化选定代码   |
| 10   | ctrl+k ctrl+0 | 折叠代码         |
| 11   | ctrl+k ctrl+j | 展开代码         |
