# SCM

| 缩写                  | 等价写法  | 描述   |
| --------------------- | --------- | ------ |
| *@daily* *@midnight*  | 0 0 * * * | 每天   |
| *@hourly*             | 0 * * * * | 每小时 |
| *@monthly*            | 0 0 1 * * | 每月   |
| *@weekly*             | 0 0 * * 0 | 每周日 |
| *@yearly* *@annually* | 0 0 1 1 * | 每年   |
